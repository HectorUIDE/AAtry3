Project Description...
Electronic Book Management System is a compact, intuitive program for creating personal and/or public  collections of your favorite books

Functionalities...
-Add book(s)
-View book collection
-Add category(ies)
-View category collection
-Search books according to their titles/author/category
-Save file (remember book collection created during the session)
-Load file (recover book collection from previous session)
-Remove book
-Edit book
